# AlphaForge — 综合投资分析系统

> 工作目录：`.` （项目根目录）

## 目录结构

```
AlphaForge/
├── 思路大纲.md      ← 用户写顶部判断，其他 AI 自动更新
├── 策略配置.md      ← 过滤条件
├── 持仓.md          ← 用户维护的持仓记录，避免重复推荐（已脱敏）
├── scripts/         ← AKShare 数据采集脚本
│   ├── run_all.py         ← 一键执行：python scripts/run_all.py
│   ├── fetch_macro.py     ← 宏观数据（CPI/PPI/M2/LPR）
│   ├── fetch_sectors.py   ← 板块数据（资金流向/涨跌幅排名）
│   └── fetch_candidates.py ← 候选股财务数据
├── data/
│   └── latest/      ← AKShare 采集缓存（综投前刷新，.gitignore排除）
├── examples/        ← 示例报告
└── wiki/            ← Obsidian 知识库（可选，需自行配置路径）
```

## 工作流

### 综投（综合投资分析）

当用户输入 `综投` + 判断/指令时：

**核心流程：理解你的判断 → 读取上次报告 → [AKShare数据]读取结构化数据 → 宏观环境小结 → 趋势变化检测 → [并行]板块动量+海外映射+负向信号 → 深挖产业数据+预期差分析 → 催化时间轴 → [多轮对抗辩论]正反方4轮对抗 → AI综合共识/分歧 → AI推荐板块 → 候选池6只 → wiki交叉验证 → 三策略每板块3只→🏆1只 → 跨板块最终推荐 → 自检 → 同步更新思路大纲+策略配置 → 输出**

1. **理解你的判断** — 作为线索参考
2. **读取** `思路大纲.md`
3. **读取** `持仓.md`
4. **读取** `策略配置.md`
5. **读取上次报告** — 读取 `examples/` 或用户指定目录下最近一期报告
6. **读取AKShare结构化数据**（若 `akshare_enabled=true`）— 读取 `data/latest/` 下所有CSV文件
7. **宏观环境小结** — 流动性/风险偏好/市场风格，先读AKShare数据，再用WebSearch补定性信息
8. **趋势变化检测**（对比上次报告）
9. **[并行] 三个独立维度扫描**：
    - **扫描板块动量**（先用AKShare板块资金流向数据，再用WebSearch补原因和分析）
    - **海外映射检测**（美股/港股领先信号）
    - **负向信号扫描**（先用AKShare板块跌幅排名数据，再用WebSearch补利空详情）
10. **深挖产业数据**（先用AKShare候选股财务数据，再用WebSearch补产业新闻）
11. **整理板块催化时间轴**（1-2个月）
12. **[多轮对抗辩论] 正反方4轮对抗** — 正方立论→反方反驳→正方回应→反方最终反驳
13. **AI综合共识/分歧** — 提炼高确信区、分歧区、风险修正，调整确信度
14. **综合以上全部信息+辩论结论，AI推荐板块**（基于数据独立决策）
15. **每个板块输出6个候选个股**，含信心等级
16. **读取 wiki 交叉验证** — 验证候选个股逻辑
17. **三策略每板块推荐3只** → 🏆 每板块1只
    - 追最强：板块内涨幅龙头或资金龙头，未来1个月有明确催化，排除PE>300x纯概念
    - 高弹性博弈：50-500亿市值，刚过拐点，弹性来自经营杠杆/产能释放/价格弹性
    - 弹性+安全边际：PE 15-30x，细分行业前3有壁垒，逻辑半衰期>6个月，经营现金流为正
18. **跨板块最终推荐**：追最强2只 + 高弹性3只 + 安全边际4只
19. **自检** — 逐一验证是否符合策略配置筛选条件
20. **同步更新思路大纲+策略配置**
21. **保存** `.md` + `.html`
22. **返回** 摘要

### 复盘

1. 读取 `examples/` 下最近3-5篇报告
2. 对比每篇推荐个股的后续实际走势（WebSearch查最新股价）
3. 检查催化事件是否兑现
4. 检查推荐逻辑是否仍然成立
5. 分策略统计胜率
6. 输出复盘表
7. 提炼经验

## 输出模板

报告包含13个章节：逻辑总图 → 判断验证 → 宏观环境 → 产业参考 → 候选股池 → 催化时间轴 → 三策略推荐 → 辩论摘要 → 跨板块推荐 → 交叉信号 → wiki引用 → 风险提示 → 验证清单

## 贯穿逻辑

所有策略贯彻 **买预期，卖现实**（Buy the rumor, sell the news）：
- **追最强**：多维度动量最强。板块业绩增速排名前5，选涨幅/资金龙头。排除PE>300x纯概念、连续亏损无收入增长、日均成交<5亿
- **高弹性博弈**：预期差最大时买入。小市值50-500亿+高经营杠杆+刚过拐点。弹性来源：经营杠杆/产能释放/价格弹性
- **弹性+安全边际**：预期形成期买入。好行业+好公司+合理价格(PE 15-30x)，细分行业前3有壁垒，逻辑半衰期>6个月，经营现金流为正
